# MLPFiM-song-subtitles
为MLP:FiM中的歌曲制作的带有逐词效果的字幕  
Karaoke-style subtitles for songs from MLP:FiM

简介
--
均为ass格式，带有逐词效果，部分字幕有更高级的特效  
制作字幕使用的音频来源多样，你可能需要自行整体平移时间轴以匹配你使用的视频或音频

关于CC BY SA 4.0内容许可证
--
一些字幕的中文部分修改自潮汐字幕社的字幕，它们的字幕采用该协议  
而完全由我制作的字幕，我也采用该协议

具体内容在字幕文件内，以注释的方式标明
